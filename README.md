# DATA CHALLENGE QRT-2021

## TECHNIQUES USED
1. PLS (PARTIAL LEAST SQUARE) REGRESSION
2. LINEAR REGRESSION

## Dependencies
```bash
pip install -r requirements.txt
```

## Usage
```bash
python run.py
```

## NOTES
  1. It assumes the train data*  and test data** files to be in the current directory. 

## 
(*) -> (X_train.csv, y_train.csv)  
(**) -> (X_test.csv)
